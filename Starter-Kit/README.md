## Compilation Instructions

g++ main.cpp pf/helper.cpp pf/gameboard.cpp pf/characters.cpp -o a

## User Manual

Commands
1. up - Move Up
2. down - Move Down
3. left - Move Left
4. right - Move Right
5. arrow - Change arrow direction
6. help - Display command list
7. quit - Quit Game
